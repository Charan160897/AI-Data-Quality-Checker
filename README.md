# AI Data Quality Checker

## Overview

AI Data Quality Checker is a Python application built with Streamlit that analyzes CSV datasets and provides detailed quality reports.

---

## Features

- CSV Upload
- Missing Value Detection
- Duplicate Detection
- Outlier Detection
- Constant Column Detection
- Data Type Summary
- Correlation Matrix
- Dashboard Metrics
- AI Dataset Summary
- Data Cleaning Preview
- Search Columns
- Analysis History
- Quality Trend Chart
- PDF Report
- CSV Report
- Logging
- Unit Testing

---

## Technologies

- Python
- Pandas
- Streamlit
- Matplotlib
- ReportLab
- Pytest
- Git

---
